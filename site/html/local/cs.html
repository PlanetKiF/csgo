Mappack (Custom Maps)

Auf dem Server werden auch Custom Maps gespielt. Dazu werden die Maps automatisch durch den Client vom Steam Workshop heruntergeladen. Dazu nutzt der Server die Planet KiF Map Collecton. Wenn Du die Maps dieser Collection schon vorab zu Deinen Favourites hinzufügst, dann lädt Dein Client sie schon vor dem ersten Besuch auf dem Server herunter, und Du musst weniger warten.

Es geht aber auch on-the-fly echt flott. Wenn es mal haken sollte: Console öffnen und retry eingeben.
